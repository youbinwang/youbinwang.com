# Aid Master

**Serious Game / Educational** · Solo Development · Unity · 2022

## Concept & Research

Aid Master teaches first aid knowledge through interactive gameplay. Combines live-action video scenarios with Kinect motion-sensing for immersive learning.

## Gameplay Design

Each level follows a structured pedagogical sequence:

1. Establish narrative context
2. Match situational elements
3. Display tools
4. Restrict certain actions
5. Conduct treatment procedures
6. Provide feedback and score settlement

## Kinect Interaction

Core interactions rely on Kinect motion-sensing technology. The system compares player actions against a pre-recorded action library to assess accuracy.

### Learning Outcomes

- Guide and feedback system design
- Motion-sensing technology (Kinect)
- Serious game design principles
- Storyboard development
