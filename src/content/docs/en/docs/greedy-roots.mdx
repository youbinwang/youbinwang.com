# Greedy Roots

**2.5D Action / Strategy** · Lead Gameplay Designer & Producer · Unity · GGJ 2023

## Game Concept (GGJ Theme: Roots)

Global Game Jam 2023 entry. Trees become corrupted by harmful fertilizers — players control a hero coconut tree to save the ecosystem.

## Core Mechanics

### Root Movement System

Players navigate by extending and retracting roots. Pulling out roots consumes nutrients and reduces height; rooting into soil restores nutrients but prevents movement.

### Resource Management

A leaf-based UI tracks nutrient levels. Full depletion means death. Sun collectibles permanently increase max health.

### Combat

Players absorb nutrients from enemies by rooting nearby. Victory requires growing taller than opponents. Rooting near stronger enemies causes nutrient loss.

### Strategic Depth

Height control determines terrain passage. Players can force enemy movement using passages unavailable to taller foes.

## Level Design

Combines resource allocation and spatial planning — players make strategic choices with limited nutrients.
