# Baihua Pavilion

**3D Multiplayer Action PvP** · Gameplay / Combat Designer · Unity · 2021

## Combat System Design

A 3D multiplayer online action PvP game with combat system design focused on multi-character balance and hit feedback.

## Character Design

Different characters have unique skill sets and combat styles. Players choose characters and strategies based on opponents.

## Multiplayer

Supports online multiplayer PvP with network synchronization and latency compensation.

- **Engine**: Unity
- **Platform**: PC (Windows)
