# 百花亭 Baihua Pavilion

**3D 多人联机动作对战** · 玩法 / 战斗策划 · Unity · 2021

## 战斗系统设计

百花亭是一款 3D 多人联机动作对战游戏，战斗系统设计注重多角色平衡和打击反馈。

## 角色设计

游戏中的不同角色拥有各自独特的技能组合和战斗风格，玩家需要根据对手选择合适的角色和策略。

## 多人联机

支持多人在线对战，实现了网络同步和延迟补偿等技术要点。
