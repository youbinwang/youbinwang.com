# Stars Chat 群星闪耀时

**叙事手游** · 主玩法与叙事设计 · Unity 2D · 2021

## 游戏概念

玩家扮演作家斯蒂芬·茨威格，通过一款名为 "Stars Chat" 的即时通讯应用与历史人物对话。灵感来源于茨威格的《人类群星闪耀时》。

核心玩法是"亲身经历决定性时刻，帮助主角解决问题"，从而获得撰写他们故事的许可。

## 叙事设计

### 角色研究

每个历史人物都经过深入的历史研究，对话风格反映其真实性格和时代特征。

### 对话撰写

编写了与历史人物性格吻合的对话，注重历史准确性与游戏趣味性的平衡。

## UI/UX

采用即时通讯界面设计，模拟现代社交应用的交互模式，让玩家以熟悉的方式与历史人物互动。

### 任务系统

设计了任务流程和道具系统，玩家通过完成任务推进与历史人物的关系。

### 项目成果

- 核心游戏概念设计
- 角色研究与对话撰写
- 任务/道具系统设计
- 在校毕业展中展出
