# Aid Master

**严肃游戏 / 教育** · 独立开发 · Unity · 2022

## 概念与研究

Aid Master 是一款严肃游戏，旨在通过互动式玩法教授急救知识。游戏结合真人视频场景增强沉浸感，配合动作感应输入设备（Kinect）实现创新交互。

## 玩法设计

每个关卡遵循结构化的教学流程：

1. 建立叙事情境
2. 匹配场景元素
3. 展示工具
4. 限制特定操作
5. 执行治疗程序
6. 提供反馈与分数结算

## Kinect 交互

游戏的核心交互依赖 Kinect 动作感应技术。系统将玩家动作与预录动作库进行比对，评估操作的准确性。这种交互方式让玩家能够直觉性地学习急救动作。

### 学习成果

- 引导与反馈系统设计
- 动作感应技术实现（Kinect）
- 严肃游戏设计原则
- 分镜脚本开发
