# Greedy Roots

**2.5D 动作 / 策略** · 主玩法设计 & 制作人 · Unity · GGJ 2023

## 游戏概念 (GGJ 主题: Roots)

Global Game Jam 2023 作品。树木因接触有害肥料而被腐化，玩家控制一棵英雄椰子树，拯救生态系统。

## 核心机制

### 根系移动系统

玩家通过伸展和回收根系进行移动。拔出根系消耗养分并降低高度；扎入土壤恢复养分但无法移动。

### 资源管理

叶片 UI 追踪养分水平。养分耗尽即死亡。太阳收集物永久增加最大生命值。

### 战斗机制

玩家通过在敌人附近扎根来吸收其养分。胜利条件是长得比对手更高。反之，在更强大的敌人旁边扎根会导致养分流失。

### 策略深度

高度控制决定了玩家能否通过地形障碍。玩家可以利用矮小敌人无法通过的通道来迫使敌人改变路线。

## 关卡设计

关卡设计结合了资源分配和空间规划，玩家需要在有限的养分资源中做出战略选择。
