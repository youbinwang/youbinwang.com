# The Journey & Drinks of Elliot Fig

**2.5D 叙事 RPG** · 主玩法与叙事设计 · Unity · 2023

## 叙事设计

Elliot Fig 是一款叙事驱动的 RPG，将调酒机制与故事叙述深度结合。玩家通过制作饮品来推进故事和影响角色关系。

### 核心设计能力

- 游戏宏观脚本撰写
- 游戏流程设计
- 媒体资产清单管理
- 纸面原型验证

## 玩法系统

### 调酒机制

核心玩法围绕饮品制作展开，不同的配方组合会触发不同的叙事分支和角色反应。

### 对话系统

使用 Yarn Spinner 实现对话系统，支持复杂的分支对话和条件触发。

## 游戏流程设计

游戏流程设计注重玩家引导和叙事节奏，通过纸面原型反复测试和迭代优化。

### 开发工具

- Unity 2.5D
- Yarn Spinner（对话系统）
- 纸面原型
- 玩测与迭代
