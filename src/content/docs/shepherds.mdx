# Shepherds

**3D 双人合作动作冒险** · 研发制作人 · 2024 · USC Games MFA 毕设

## 游戏概述

Shepherds 是一款 3D 双人合作动作冒险游戏。两位不太可能的搭档——牧羊人 Zoe 与狼 Kin——需要结合各自独特的能力，克服节奏解谜和物理平台跳跃的挑战，深入活火山的中心。

### 角色

**Zoe（牧羊人）** — 渴望成为像已故母亲一样的专业牧羊人，始终相信每个人内心的善良。

**Kin（狼）** — 试图探寻自己的起源，不确定为什么一只狼会存在于岛上。不信任他人，自力更生。
